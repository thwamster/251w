    \noindent Problem 2.3.5 
    \vspace{0.3cm}
    
    \underline{proposition:} Let $a, b,$ and $c$ be integers.
    If there exists an integer $d$ such that $d|a$ and $d|b$ but $d\not|c$, then $ax+by=c$ has no integer solutions for $x$ and $y$.
    \vspace{0.4cm}
    
    \fbox{proof} (Contradiction)
    \vspace{0.2cm}
    
    Let $a$, $b$, and $c$ be arbitrary integers.
    Let $d$ be a particular integer so that $d|a$, $d|b$, and $d\not | c$.
    Let $x$ and $y$ be particular integers so that $ax + by = c$.
    We will show this leads to a contradiction.
    By the properties of integer divisibility, let $f$ and $g$ be particular integers so that $a = df$ and $b = dg$.
    By substitution, we see $(df)x + (dg)y = c$.
    By the distributive law, we see $d(fx + gy) = c$.
    Since addition is closed on the integers, let $h$ be a particular integer so that $h = fx + gy$.
    By substitution, we see $dh = c$.
    By the properties of integer divisibility, we see $d | c$.
    This contradicts with $d \not | c$.
    Therefore, for all integers $a$, $b$, and $c$, if there exists an integer $d$ so that $d|a$, $d|b$, and $d\not | c$, there does not exist integers $x$ and $y$ so that $ax + by = c$.
    $\blacksquare$
